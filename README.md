Modo de uso:

1. Ejecutar la función MAIN (src/main/java/analizadorLexico/Main.java) y esperar unos segundos.
2. Aparecerá una interfaz en la cual se podrá abrir el archivo de prueba deseado (o mismo crearlo sobre la interfaz). Se recomienda utilizar el archivo "prueba.txt" presente en la raíz del proyecto, pues abarca todos las funcionalidades y principalmente ahorra tiempo.
3. Luego de cargar el archivo, en la misma solapa donde se encuentra el botón para cargarlo existe un botón "Analizar". Una vez presionado, se hará el análisis del archivo.
4. Luego del análisis, en el archivo tabla_de_simbolos.txt podrá verse la tabla generada, correspondiente al archivo en cuestión.
